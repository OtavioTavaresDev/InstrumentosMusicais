🎸 guitarraVirtual

Uma guitarra virtual interativa criada com HTML, CSS e JavaScript, com visual moderno, iluminação estilizada e geração de som em tempo real direto no navegador.

O projeto simula cordas de guitarra que podem ser tocadas digitalmente, oferecendo uma experiência visual imersiva e interação musical instantânea.

✨ Funcionalidades

🎸 Simulação de cordas de guitarra

🖱️ Interação por clique/toque nas cordas

🎛️ Controles de modos/efeitos sonoros (via botões)

🔊 Som gerado em tempo real (síntese digital)

🌌 Visual escuro com estética moderna

💡 Botões com estados ativos e feedback visual

📱 Compatível com desktop e dispositivos móveis

🧠 Tecnologias usadas
Tecnologia	Função
HTML5	Estrutura da interface
CSS3	Design da guitarra, cordas e efeitos visuais
JavaScript	Interação, lógica sonora e controles
Web Audio API	Geração e manipulação do som
🎵 Como o som funciona

O áudio não é reproduzido por arquivos. Ele é criado digitalmente:

Oscilador → Ajuste de frequência (nota) → Controle de volume → Saída


Cada corda:

Possui frequência própria

Pode aplicar variações dependendo do modo selecionado

Usa envelope de som para evitar cortes bruscos

🚀 Como usar

Abra o arquivo guitarraVirtual.html no navegador

Clique ou toque nas cordas para tocar

Use os botões de controle para alterar o comportamento do som
